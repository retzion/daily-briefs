## Featured: `/xlsx` Skill (Spreadsheet Operations)

Claude Code's `/xlsx` skill handles spreadsheet files directly in your coding session — no need to switch to Excel or write pandas boilerplate.

**What it covers:**
- **Read/edit** existing `.xlsx`, `.xlsm`, `.xltx`, `.csv`, `.tsv` files — add columns, compute formulas, format cells, create charts
- **Create** new spreadsheets from scratch or from other data sources (JSON, API responses, database exports)
- **Convert** between tabular formats (CSV to XLSX, TSV to XLSX, etc.)
- **Clean** messy data files — fix malformed rows, misplaced headers, junk data

**When to use it:** Any time a spreadsheet file is the primary input or output. The skill triggers automatically when you reference a spreadsheet file by name or path.

**Example:**
```
> /xlsx Read sales_q2.xlsx, add a "YoY Growth %" column comparing to Q2 last year, 
  highlight any negative growth in red, and export as both .xlsx and .csv
```

Claude reads the file, computes the growth percentages, applies conditional formatting, and writes both output files — all without leaving your terminal.
