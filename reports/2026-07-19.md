## Featured: MCP Auto-Background

New in Week 29 (v2.1.207+): MCP tool calls that run longer than two minutes now move to the background automatically, keeping your session responsive instead of blocking on a slow external integration.

**Why it matters**: MCP servers connect Claude Code to databases, APIs, browsers, and other services — but some operations (large queries, complex API calls, browser automation) can take minutes. Previously, a slow MCP call would lock the session. Now Claude keeps working while the call completes.

**Configuration**:
```bash
# Change threshold to 90 seconds (in milliseconds)
export CLAUDE_CODE_MCP_AUTO_BACKGROUND_MS=90000

# Disable auto-backgrounding
export CLAUDE_CODE_MCP_AUTO_BACKGROUND_MS=0
```

No setup required — the 2-minute default is active out of the box. Background MCP calls surface their results when complete, just like background subagents.
