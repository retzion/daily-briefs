## Featured: Screen Reader Mode (`--ax-screen-reader`)

New in v2.1.208, Claude Code now has a dedicated screen reader mode for visually impaired developers. It strips all decorative TUI elements (spinners, progress bars, color codes, box-drawing characters) and renders output as clean, sequential plain text that works well with Voiceover, NVDA, JAWS, and other assistive technology.

**Why it matters:** Agentic coding tools have historically been inaccessible — rich terminal UIs that look great but are opaque to screen readers. This is a meaningful step toward making AI-assisted development available to everyone.

**How to use it:**

```bash
# Flag
claude --ax-screen-reader

# Environment variable
export CLAUDE_AX_SCREEN_READER=1

# Persistent setting (settings.json)
{ "axScreenReader": true }
```

All tool outputs, diffs, and progress updates render as structured plain text. Works in both interactive and headless modes.
