## Featured: ant CLI

The `ant` CLI is Anthropic's new official Go-based command-line client for the Claude API, launched April 2026. Unlike Claude Code (your interactive coding assistant), `ant` is a programmatic API client — every API resource is exposed as a subcommand with output formatting, response filtering, and YAML/JSON file input.

**Why it's useful:** It bridges the gap between Claude Code (interactive) and raw API calls (verbose). Useful for scripting, CI pipelines, managing Managed Agents, and quickly testing prompts without writing code.

**Quick start:**
```bash
brew install anthropics/tap/ant        # macOS
export ANTHROPIC_API_KEY=sk-...
ant messages create --model claude-sonnet-4-6 \
  --max-tokens 1024 \
  --message "role=user,content=Hello"
ant --help                             # list all subcommands
```

[Documentation](https://platform.claude.com/docs/en/api/sdks/cli) | [GitHub](https://github.com/anthropics/anthropic-cli)
