## Featured: Stacked Slash-Skill Invocations

New in v2.1.199, Claude Code now supports **chaining multiple skills in a single prompt**. Previously, typing `/skill-a /skill-b` would only load the first skill. Now it loads all leading skills (up to 5) before processing your request.

**Why it's useful:** Many workflows benefit from combining skills — for example, loading design guidance before generating a visualization, or running a security review alongside a code review. Instead of invoking skills one at a time across multiple turns, you can now front-load all the context you need in a single command.

**Quick usage:**

```
> /dataviz /code-review Build a metrics dashboard and review the implementation
> /security-review /simplify Audit and clean up the auth middleware
> /deep-research /init Research best practices for monorepo tooling and set up CLAUDE.md
```

Each skill loads its guidance into context in order, then your prompt is processed with all of them active. The 5-skill cap prevents context bloat — in practice, 2–3 stacked skills covers most workflows.
