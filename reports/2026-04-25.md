## Featured: Custom Themes (`/theme`)

Claude Code v2.1.118 introduced a full custom theme system, letting you personalize your terminal UI beyond the built-in presets.

**What it is:** A slash command that lets you create, switch between, and manage named color themes for Claude Code's terminal interface.

**Why it's useful:** Long coding sessions are easier on the eyes with colors tuned to your preference. Teams can standardize aesthetics by shipping themes via plugin `themes/` directories.

**Quick usage:**
```
/theme
# Select "New custom theme..." from the menu
# Name your theme and pick color tokens to override
```

Themes are stored as JSON in `~/.claude/themes/` and can be hand-edited:
```json
// ~/.claude/themes/dark-ocean.json
{
  "primary": "#5eead4",
  "secondary": "#818cf8",
  "background": "#0f172a"
}
```

Plugins can also bundle themes by including a `themes/` directory, making team-wide sharing simple.
