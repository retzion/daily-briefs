## Featured: Plugin Marketplace Token Estimates
The May 15 release added **projected context cost** to the `/plugin` marketplace browse pane. Before installing a plugin, you now see two numbers:

- **Per-turn cost** — how many tokens the plugin's tool schemas and injected context add to every turn
- **Per-invocation cost** — estimated tokens consumed each time the plugin's tools are actually called

This matters because MCP plugins can silently bloat your context. A plugin with 20 tools might inject 25K+ tokens of schema per turn even if you never use it. Now you can comparison-shop plugins by their context overhead before committing.

**Usage**: Run `/plugin` → browse the marketplace → look for the token estimates displayed next to each plugin listing. Use this to make informed decisions about which plugins are worth their context budget.
