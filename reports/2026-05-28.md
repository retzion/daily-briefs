## Featured: Status Line Customization

The Claude Code **status line** is a configurable bar at the bottom of your terminal that runs any shell script you specify. Today's v2.1.153 release enhanced it with `COLUMNS` and `LINES` environment variables so scripts can size output to your terminal width.

**Why it's useful:** Get persistent, at-a-glance visibility into context usage, costs, git branch, rate limits, or anything else — without interrupting your workflow.

**Quick setup:**

Use `/statusline` to have Claude generate one for you, or configure manually in `~/.claude/settings.json`:

```json
{
  "statusLine": {
    "type": "command",
    "command": "sh ~/.claude/statusline.sh",
    "padding": 0
  }
}
```

Your script receives JSON on stdin with model info, context window %, cost data, workspace path, and rate limits. Output supports ANSI colors and multiple lines. Community tools like [ccstatusline](https://github.com/sirmalloc/ccstatusline) offer 30+ pre-built widgets with Powerline-style themes.

[Docs](https://code.claude.com/docs/en/statusline)
