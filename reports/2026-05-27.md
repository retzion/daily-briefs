## Featured: MessageDisplay Hook

The new `MessageDisplay` hook event (v2.1.152) lets hooks **transform or hide assistant message text** as it's displayed in your terminal. This is a display-layer hook — it doesn't change what the model sees or what's stored in the transcript, only what you see.

**Why it's useful:** Teams can redact sensitive output (API keys, connection strings) before it hits the screen, reformat responses to match house style, inject visual annotations, or suppress verbose boilerplate. It's the first hook that operates on the assistant's output stream rather than on tool calls or session lifecycle events.

**Quick example** — a hook that redacts AWS account IDs:

```json
{
  "hooks": {
    "MessageDisplay": [
      {
        "command": "sed 's/[0-9]\\{12\\}/[REDACTED]/g'",
        "timeout": 1000
      }
    ]
  }
}
```

Add this to `.claude/settings.json`. Every assistant message will pass through the hook before rendering; 12-digit numbers (matching AWS account ID format) are replaced with `[REDACTED]`.

**Other use cases:** strip ANSI color codes for accessibility, pipe output through `jq` for JSON-heavy sessions, or add custom prefixes/timestamps to responses.
