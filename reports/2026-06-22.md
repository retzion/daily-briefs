## Featured: `/powerup` (Interactive Lessons)

`/powerup` is Claude Code's built-in interactive tutorial system, shipping 18 guided lessons with animated terminal demos. It covers everything from beginner fundamentals (context management, permission modes) to advanced workflows (agent teams, dynamic workflows). Each lesson runs 3–10 minutes, right in your terminal.

**Why it's useful:** Claude Code has a massive feature surface — skills, hooks, plugins, MCP servers, auto mode, subagents. Most users only discover features when they stumble on them. `/powerup` solves the discovery problem with hands-on walkthroughs instead of docs.

**Usage:**
```
claude
> /powerup
# Browse the 18 lessons, pick one, follow along
# No plan restrictions — available to all users
```

Available since v2.1.90. No internet connection required; the lessons run locally.
