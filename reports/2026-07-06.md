## Featured: `/claude-api` Skill (Claude API Quick Reference)

The `/claude-api` skill is a built-in Claude Code reference that provides up-to-date Claude API documentation — model IDs, pricing, parameters, streaming, tool use, MCP, agents, caching, and token counting — without leaving your session. It triggers automatically when your prompt mentions Claude, Anthropic, or LLM-related topics, but you can also invoke it directly.

**Why it's useful:** When building AI applications with the Claude API, you often need to look up the latest model ID, check pricing, or get the correct tool-use schema. Instead of switching to a browser, `/claude-api` loads the current reference into context so Claude Code can give accurate, current answers.

**Quick usage:**
```bash
# Explicit invocation
> /claude-api What's the latest model ID for Sonnet and its pricing?

# It also auto-triggers when you ask LLM-shaped questions:
> "How do I set up streaming with tool use in the Anthropic SDK?"
# → skill loads automatically before Claude responds

# Great for migration tasks:
> /claude-api I'm upgrading from Sonnet 4.6 to Sonnet 5 — what breaks?
```

The skill skips loading when it detects you're working with a non-Anthropic provider (OpenAI, Gemini, etc.), so it won't interfere with other LLM work.
