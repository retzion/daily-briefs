## Featured: Keybindings Customization (`~/.claude/keybindings.json`)

Claude Code supports fully customizable keyboard shortcuts via a JSON config file. This is useful for power users who want to remap submit keys, add chord shortcuts, or adapt Claude Code to match muscle memory from other editors.

**How to set it up:**
1. Run `/keybindings` in Claude Code to create or open `~/.claude/keybindings.json`
2. Add bindings scoped to specific contexts (Chat, Editor, App)
3. Changes are hot-reloaded — no restart needed

**Example — remap Ctrl+Enter to submit and add a chord shortcut:**
```json
{
  "bindings": [
    {
      "context": "Chat",
      "keys": {
        "ctrl+enter": "chat:submit",
        "ctrl+k ctrl+r": "chat:resetConversation"
      }
    },
    {
      "context": "App",
      "keys": {
        "ctrl+t": "app:toggleTodos"
      }
    }
  ]
}
```

Key details:
- Each binding belongs to a **context** (Chat, Editor, App) that controls when it fires
- Actions use `namespace:action` format (e.g., `chat:submit`, `app:toggleTodos`)
- Chord shortcuts (multi-key sequences like `ctrl+k ctrl+r`) are supported
- Vim mode compatible — most `Ctrl+key` combos pass through even in NORMAL mode
- Run `/doctor` to check for keybinding warnings

Docs: [Customize keyboard shortcuts](https://code.claude.com/docs/en/keybindings)
