## Featured: Security-Guidance Plugin

The **security-guidance plugin** is a free, three-stage security scanner that ships as an official Claude Code plugin. Unlike `/security-review` (a one-shot skill you invoke manually), this plugin runs continuously throughout your session.

**How it works — three review checkpoints:**
1. **File edits** — fast deterministic pattern match (no model call) flags dangerous constructs: `eval()`, `os.system()`, `child_process.exec()`, `pickle` deserialization, `.innerHTML=`, `dangerouslySetInnerHTML`
2. **Model turns** — Claude analyzes the session's full git diff for authorization bypass, IDOR, injection flaws, SSRF, and weak crypto
3. **Commits/pushes** — deepest review examines surrounding files, sanitizers, and related code paths to validate findings and reduce false positives

**Install it:**
```
/plugin install security-guidance@claude-plugins-official
/reload-plugins
```

**Why it matters:** Anthropic reports a 30–40% reduction in security-related PR comments in internal use. It shifts security left without adding friction — findings surface inline as you code, not after you open a PR.
