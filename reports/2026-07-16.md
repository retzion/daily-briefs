## Featured: `--forward-subagent-text` (Subagent Stream Forwarding)

New in v2.1.211, the `--forward-subagent-text` flag exposes subagent text and thinking in Claude Code's stream-json output — making multi-agent workflows observable from the outside.

**Why it matters:** When Claude Code spawns subagents (via the Agent tool or workflows), their output was previously opaque to external consumers. If you're building a CI pipeline, custom dashboard, or wrapper tool on top of Claude Code, you had no visibility into what subagents were doing. This flag changes that.

**Quick usage:**
```bash
# CLI flag
claude --output-format stream-json --forward-subagent-text -p "refactor auth module"

# Or via environment variable
export CLAUDE_CODE_FORWARD_SUBAGENT_TEXT=1
claude --output-format stream-json -p "refactor auth module"

# Subagent text and thinking now appear as events in the JSON stream,
# letting you build progress UIs, log agent reasoning, or pipe to monitoring tools.
```

Especially useful for CI/CD integrations and the Claude Code SDK, where understanding what's happening inside orchestrated agent workflows is critical for debugging and observability.
