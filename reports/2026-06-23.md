## Featured: Claude Code Artifacts

Claude Code Artifacts (beta, June 18) let you turn any coding session into a self-contained, interactive HTML page published to a private URL on claude.ai. Unlike regular artifacts in Claude.ai chat, these pull from your full session context — codebase, MCP connectors, and conversation — to produce rich pages like incident reports, architecture explainers, or release checklists.

**Why it's useful:** Instead of copy-pasting terminal output into a doc or building a one-off dashboard, you can ask Claude Code to "make an artifact" and get a live page that teammates can view immediately. When the session updates the artifact, the page refreshes in place with full version history.

**Quick usage:**
```
> Create an artifact summarizing this PR with the diff, test results, and deployment checklist
```

Claude Code builds the page and returns a shareable link. Only authenticated org members can view it. Available on Team and Enterprise plans.
