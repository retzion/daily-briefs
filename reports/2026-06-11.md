## Featured: `/context` (Context Window Inspector)

The `/context` slash command gives you a live breakdown of what's consuming your context window, with token counts and percentages for each category:

- **System prompt** — base instructions
- **System tools** — tool definitions loaded for the session
- **Custom agents** — any spawned subagent definitions
- **Memory files** — CLAUDE.md and auto-memory content
- **Messages** — your conversation history, file reads, and tool results
- **Free space** — remaining capacity

**Why it's useful**: When Claude starts feeling slow or missing information you shared earlier, `/context` tells you exactly where your tokens went. It's the first diagnostic to run before reaching for `/compact`.

**Quick usage**:
```
> /context

Context usage:
  System prompt:    4,200 tokens  (2.1%)
  System tools:    12,800 tokens  (6.4%)
  Memory files:     1,200 tokens  (0.6%)
  Messages:       145,000 tokens (72.5%)
  Free:            36,800 tokens (18.4%)
```

If messages are eating 70%+ of context, run `/compact` to compress the conversation. If memory files are unexpectedly large, audit your CLAUDE.md files.
