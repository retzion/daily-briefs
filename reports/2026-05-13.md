## Featured: Agent View (`claude agents`)

Shipped in v2.1.139 as a Research Preview, **Agent View** gives you a single dashboard to see every Claude Code session — running, blocked on input, or completed — across your machine.

**Why it's useful:** As agentic workflows scale up, developers often have multiple Claude Code sessions running in parallel (background tasks, worktree subagents, long-running builds). Agent View centralizes visibility so you can spot blocked sessions that need input, check on background work, and jump into any session without hunting through terminal tabs.

**Usage:**
```bash
# Launch Agent View from the terminal
claude agents

# Or press the left arrow key from any active session to open it
```

Each session shows its status, elapsed time, and token usage. Select any session to jump into its transcript. As the feature matures, expect tighter integration with Routines and Remote Control for full fleet management of your coding agents.
