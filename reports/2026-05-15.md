## Featured: Fast Mode (`/fast`)

Fast mode makes Claude Opus respond up to 2.5x faster in Claude Code — same model, same intelligence, higher throughput. As of v2.1.142 (yesterday), it defaults to **Opus 4.7**.

**Why it matters:** Long agentic sessions with many tool calls benefit significantly from faster output. The speed-up compounds across dozens of agent turns — a 10-minute refactoring task can drop to ~4 minutes.

**How to use it:**
```
# Toggle in-session
/fast          # press Tab to toggle on/off — ↯ icon appears when active

# Enable permanently in settings
# ~/.claude/settings.json
{ "fastMode": true }

# Or via environment variable for Opus 4.6 fallback
CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1
```

**Cost:** 6x standard Opus pricing per token. Best used for complex, multi-step tasks where wall-clock time matters more than token cost. ([Docs](https://code.claude.com/docs/en/fast-mode))
