## Featured: Streaming Idle Watchdog
New in v2.1.196 and **enabled by default**. When a streaming response from the model stalls — no tokens received for 5 minutes — Claude Code automatically aborts the request and retries.

**Why it matters:** Long-running sessions, background agents, and multi-agent workflows occasionally hit silent stalls (network hiccups, transient API issues, model overload). Previously, these would hang indefinitely, requiring manual intervention. The watchdog makes Claude Code self-healing in these scenarios.

**Usage:** No configuration needed — it's on by default in v2.1.196+. If you're running background agents or overnight routines, this significantly improves reliability without any action on your part.
