## Featured: `language` Setting (Session Title Localization)

Claude Code v2.1.176 introduced the `language` setting, which controls the language used for auto-generated session titles. Previously, session titles were always English regardless of the conversation language — a long-standing pain point ([issue #35799](https://github.com/anthropics/claude-code/issues/35799)).

**What it does:** When set, Claude Code generates session titles in your preferred language, making the session list immediately scannable for non-English speakers.

**How to use it:**

```bash
# Set globally
claude config set language "ja"

# Or in .claude/settings.json
{
  "language": "ja"
}
```

Supported values follow standard language codes (`ja`, `ko`, `zh`, `es`, `fr`, `de`, `pt`, etc.). The setting also influences other UI-generated text where applicable.
