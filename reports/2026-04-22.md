## Featured: Claude Code Routines

Routines (research preview, launched April 14) let you configure a Claude Code automation once — a prompt, a repo, and your connectors — then run it on a schedule, via API call, or in response to a GitHub webhook. They run on Anthropic's cloud infrastructure, so your laptop doesn't need to be open.

**Why it's useful:** Automates recurring dev tasks like nightly issue triage, weekly docs-drift scans, deploy verification, and alert triage — no local setup required.

**Quick example:**
1. Open Claude Code desktop > Routines > New Routine
2. Set trigger: Schedule > Daily at 2:00 AM
3. Set prompt: `Review all PRs merged today. Update CHANGELOG.md with a summary grouped by type.`
4. Connect your repo and GitHub connector
5. Save — it runs unattended each night

**Limits:** Pro: 5/day, Max: 15/day, Team/Enterprise: 25/day.
