## Featured: `/config key=value` (Inline Settings)

New in v2.1.181 (June 17). Set any Claude Code setting directly from the prompt without opening a menu or editing JSON files.

**Why it's useful:** Quick toggling mid-session — turn off thinking for a fast answer, switch models, adjust behavior — without breaking your flow. Works in interactive mode, headless (`-p`), and Remote Control.

**Examples:**
```
/config thinking=false
/config model=claude-sonnet-4-6
/config autoMode=true
```

Settings revert when the session ends. For persistent changes, use `/config` without arguments to open the settings menu.
