## Featured: /loop Skill
The `/loop` skill runs any prompt or slash command on a recurring interval — perfect for monitoring deploys, polling CI status, or periodic health checks.

**Usage:**
```
/loop 5m /review          # runs /review every 5 minutes
/loop 10m check deploy status  # checks deploy status every 10 minutes
/loop                     # defaults to 10-minute interval
```

Instead of manually re-running commands or setting up external cron jobs, `/loop` keeps a recurring task running inside your Claude Code session. Combine it with `/review` or custom prompts to create lightweight monitoring workflows without leaving your terminal.
