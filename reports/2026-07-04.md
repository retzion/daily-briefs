## Featured: Draft PR Handoff (Background Agent Auto-PR)

Since v2.1.198 (July 1), background agents launched from `claude agents` that work in a worktree will automatically commit, push, and open a GitHub draft PR when they finish — no manual steps between task completion and a reviewable PR.

**Why it matters:** This closes the "last mile" gap in background agent workflows. Previously, agents would complete work and then stop to ask. Now the cycle is fully autonomous up to the merge decision, which stays with a human.

**How to use it:**
```bash
# 1. Open the agents view
claude agents

# 2. Launch a background agent with a task
#    (it automatically gets a worktree)
> Refactor the auth module to use JWT validation

# 3. When done, the agent commits, pushes, and opens a draft PR
# 4. Review the draft PR and merge at your convenience
```

**To disable** (e.g., for propose-only workflows where humans control all commits), see the discussion on [GitHub issue #73197](https://github.com/anthropics/claude-code/issues/73197) for adding a settings toggle.
