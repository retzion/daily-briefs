## Featured: DesignSync Tool

The DesignSync tool enables two-way synchronization between Claude Design (Anthropic's visual design tool) and Claude Code. Shipped as part of the Claude Design June update, it bridges visual design and code implementation.

**What it does:**
- **Pull**: Import design system tokens (colors, typography, spacing, components) from a Claude Design canvas into your codebase
- **Push**: Send built components back to the design canvas, keeping design and implementation in sync

**Why it matters:** Designers and developers traditionally work from static handoffs (Figma specs, screenshots). DesignSync creates a live loop — design changes flow into code, and code changes reflect back in the canvas. This eliminates drift between design intent and shipped UI.

**Usage:**
```
> Use the DesignSync tool to pull design tokens from a linked canvas
> Push built components back after implementation
```

Requires a Claude Design canvas linked to your project. Available in beta on Pro plans and above.
