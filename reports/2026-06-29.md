## Featured: Claude Tag (Slack Agentic Integration)

**Claude Tag** is Anthropic's new Slack-native AI agent (beta, launched June 23). Unlike the previous Claude-in-Slack chatbot, Tag is a **multiplayer agentic teammate** — one Claude identity per workspace that can be @mentioned in any channel.

**What it does**: @Claude with a task and it breaks the work into stages, executes them, and responds in a thread. It can read linked files, search the web, write and run code, and chain multi-step workflows — all within Slack's native interface.

**Why it matters**: Bridges the gap between "ask Claude a question" and "assign Claude a task." Teams can delegate research, code review triage, data analysis, and documentation tasks without leaving Slack. Anthropic describes it as "the beginning of an evolution of Claude Code" — agentic capabilities moving beyond the terminal.

**Quick usage**:
```
@Claude analyze our Q2 error logs and summarize the top 5 recurring issues with recommended fixes
```
Claude breaks this into steps (fetch logs → categorize → rank → draft fixes), posts progress updates in the thread, and delivers a structured summary.

**Availability**: Enterprise and Team plans. The legacy Claude-in-Slack integration retires **August 3, 2026** — migrate before then.

**Source**: [Introducing Claude Tag](https://www.anthropic.com/news/introducing-claude-tag)
