## Featured: Shell Mode Auto-Explain (`!` prefix)

Claude Code's shell mode lets you run terminal commands directly from the conversation by prefixing them with `!`. **New this week**: output is now automatically analyzed and explained — no follow-up prompt needed.

**What it does**: Instead of switching to a separate terminal, you can run commands inline. Claude sees the output and immediately provides context — error explanations, test result summaries, or next-step suggestions.

**Why it's useful**: Eliminates the copy-paste-explain loop. Particularly handy for debugging test failures, checking build output, or exploring unfamiliar codebases where you'd normally run a command and then ask "what does this mean?"

**Quick example**:
```
! npm test -- --filter auth
```
Claude runs the tests, sees the output, and immediately explains which tests failed and why — without you having to ask. Also new in v2.1.193: **live file path autocomplete** works in `!` commands, so typing `! cat src/` will suggest completions.

**Tip**: Combine with `autoMode.classifyAllShell` (also new this week) to have all shell commands vetted by the auto-mode classifier before execution.
